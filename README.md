# Projeto Agentes Autónomos 2025/2026
## Por: Constança Ferreira (122705) e Íris Baptista (122701)
### Problemas: Farol e Foraging

Para executar este projeto, apenas precisa de:
- Editor de texto (Visual Studio Code, Pycharm...)
- Fazer download do projeto em pasta ``.zip`` (clique no botão ``Code`` e na opção ``Download ZIP``)


Após fazer download do projeto, só tem que:
1. Extrair o projeto do ``.zip``
2. No editor de texto (Visual Studio Code, Pycharm...), selecionar a opção de importar projeto e selecionar a pasta extraída
3. Executar o ficheiro ``MotorSimulator.py`` e utilizar a consola para interagir com a **TUI** para correr o programa

Obrigada!
